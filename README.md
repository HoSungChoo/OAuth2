# OAuth 2.0
OAuth 2.0 개념과 흐름을 이해하고, 스프링 환경에서 이를 적용하는 과정

OAuth 2.0 + Spring Security + JWT를 이용하여 인증/인가 및 회원 데이터를 적재하는 기능을 가진다.

## 개발 환경
- Java : 21
- Spring : 3.4.0
- MySQL : 8.0.30
- JWT : 0.12.3
